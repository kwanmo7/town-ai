# Prompt 품질 평가표

점수 기준:

- 1점: 사용할 수 없음
- 2점: 큰 수정이 필요함
- 3점: 일부 수정 후 사용할 수 있음
- 4점: 실사용에 충분함
- 5점: 매우 유용하고 신뢰할 수 있음

## 필수 확인

- [ ] 입력에 없는 지역 특성, 교통, 치안, 시세, 편의시설 또는 재해 위험을 사실처럼 단정하지 않았다.
- [ ] Backend가 제공한 점수, 평균, 순위, Area 순서를 바꾸지 않았다.
- [ ] 점수 차이가 작은 경우 우열을 과장하지 않았다.
- [ ] 방문 횟수가 적은 Area의 해석 한계를 명시했다.
- [ ] 점수와 memo가 충돌할 때 이를 숨기지 않고 재확인 필요성을 설명했다.
- [ ] AREA·COMPARE·ALL의 확인 항목이 막연한 경고가 아니라 실제 행동으로 작성됐다.
- [ ] 동일한 숫자와 문장을 불필요하게 반복하지 않았다.

## 합격 기준

- 자동 구조 검증을 모두 통과한다.
- 입력에 없는 사실을 단정한 항목이 하나도 없다.
- 적용 가능한 각 평가 항목이 3점 이상이다.
- 각 Report의 평균 점수가 4.0 이상이다.
